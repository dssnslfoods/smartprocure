# Smart Procurement System

A professional, end-to-end procurement and supplier management platform designed to streamline the purchasing workflow, enhance vendor transparency, and optimize enterprise procurement processes.

## Key Features

### 🏢 Supplier Lifecycle Management
*   **Supplier Registration & Onboarding:** Automated self-service registration for new vendors with multi-tier approval workflows.
*   **Vendor Portal:** A dedicated workspace for suppliers to manage profiles, view opportunities, and submit bids.
*   **Centralized Supplier Directory:** Comprehensive database of pre-qualified vendors with performance history.

### 📝 Request for Quotation (RFQ) & Bidding
*   **RFQ Management:** Standardized tools for creating and distributing detailed procurement requests.
*   **Electronic Bidding:** Secure platform for suppliers to submit competitive offers in real-time.
*   **Quotation Tracking:** End-to-end visibility of all bid submissions and revisions.

### 📊 Evaluation & Awards
*   **Objective Scoring Engine:** Customizable evaluation templates for comparing vendors across multiple criteria (Price, Quality, Lead Time, etc.).
*   **Side-by-side Comparisons:** Performance visualizers to aid in informed decision-making.
*   **Contract Awarding:** Streamlined process for finalizing selections and notifying successful bidders.

### 📈 Analytics & Reporting
*   **Executive Dashboards:** Real-time visibility into procurement KPIs, spending trends, and vendor performance.
*   **Comprehensive Reports:** Audit-ready documentation and strategic insights for management.

### ⚙️ Administrative Control
*   **Role-Based Access Control (RBAC):** Granular permission management for Buyers, Suppliers, and Administrators.
*   **System Configuration:** Highly configurable settings to align the system with organizational procurement policies.

---

## Technical Architecture
*   **Frontend:** React 18, TypeScript, Vite
*   **Styling:** Tailwind CSS, Shadcn UI
*   **State Management:** TanStack Query (React Query)
*   **Data & Auth:** Supabase (Backend-as-a-Service)
